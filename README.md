# Terraform Cluster of Web Servers (Tutorial)

Terraform declarations for Cluster of Web Servers (educational purpose)

### Prerequisites

Terraform v0.11.3

Check your version.

```
terraform -v

```

### Installing

Install dependencies using `brew`.

```
brew install terraform
```

## Usage

To see how to use:

```
terraform plan
terraform apply
```
